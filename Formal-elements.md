# Main Title: Stranger Danger

**tagline:** Outsmart the thieves and protect your home!

## Game Essence

The player, a child alone in a large mansion, must defend against intruding thieves by setting up traps in various rooms.

Platform: PC (Unity)

Use screenshots to illustrate your answers.

### 1. Players

* Target Audience: Ages 10+ interested in strategy and home-defense games.
* Number of Players: Single-player.
* Interaction Patterns: Player versus environment.


### 2. Goals

* Primary Goal: Prevent thieves from entering by setting up traps in different rooms.
* Goal Communication: Instructions on object interaction and alerts when thieves approach.


### 3. Processes

* Game Start: Player learns of potential threats and receives guidance on interactions.
* Core Gameplay: Move between rooms, find objects for traps, and anticipate thief paths.
* Game End: Win by preventing any thief entry, lose if thieves enter certain rooms.


### 4. Rules

* Traps require specific items to set up.
* Game alerts players of thieves’ proximity to allow time for setup.


### 5. Resources

* Trap Materials: Found or combined in rooms.
* Alerts: Notifications of nearby thieves.

### 6. Conflicts

* Main Challenge: Setting traps in time to stop thieves.


### 7. Boundaries

* Game Boundaries: The mansion confines the player, with specific items and threats in each room.


### 8. Outcomes

* Success by preventing entry, failure if thieves bypass traps.
* this game requires skills and not depends on luck
