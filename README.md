# Stranger Danger

**Description:**  
In *Stranger Danger*, you play as a child home alone in a large mansion, defending yourself against sneaky thieves. Move room to room to set up creative traps, using items in each space to keep the intruders at bay. The challenge increases with each successful defense, testing your ability to protect the home.

**Platform:**  
PC (Unity)

**Gameplay Features:**
- Explore a multi-room mansion, each with items for creating traps.
- Set up defenses in time as you anticipate thief movements.
- Choose from multiple difficulty levels for varying speeds of setup and thief detection.
- Defend successfully to keep your home safe from invaders.
